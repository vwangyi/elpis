# HTML 标签

> 写法：<标签名>内容</标签名>

## 最普通的标签（换行）5

> 多个div标签会换行
>
> 块级元素

```html
<div>内容</div>
```

## 最普通的标签（不换行）5

> 多个span标签也不换行，但是中间会有一个空格
>
> 行级元素

```html
<span>内容</span>
```

## 斜体标签 0

> 字体倾斜
>
> 多个i标签也不换行，但是中间会有一个空格
>
> 行级元素

```html
<i>内容</i>
```

## 标题标签 2

> h1到h6（从大到小）
>
> 多个h标签会换行
>
> 块级元素

```html
<h1>标题一</h1>
<h2>标题二</h2>
<h3>标题三</h3>
<h4>标题四</h4>
<h5>标题五</h5>
<h6>标题六</h6>
```

## 段落标签 1

> 多个p标签会换行，有间距
>
> 块级元素 block

```html
<p>内容</p>
```

## 链接标签 5

> 多个a标签不换行
>
> 行级元素 inline

```html
<a href="https://juejin.cn/post/7086735198942920712">百度一下</a>
```

## 图片标签 5

> 多个img标签不换行
>
> alt：图片不能正常显示的时候 此处显示的提示内容
>
> 行级元素 inline-block

```html
<img
  src="https://p3.toutiaoimg.com/tos-cn-i-qvj2lq49k0/fa2c8a613f9f47368f3b733dc4584098~tplv-tt-cs0:640:360.jpg?from=feed&_iz=31826"
  alt="hello"
/>
```

## 有序列表 2

> ol: ordered list
>
> li: list item
>
> ol下边也必须是li
>
> 多个li要换行
>
> ol: 块级元素
>
> li：块级元素

```html
<style>
  ol {
    margin: 0;
    padding: 0;
    list-style: none;
  }
</style>

<ol>
  <li>a</li>
  <li>b</li>
  <li>c</li>
  <li>d</li>
</ol>
```

## 无序列表 5

> ul: 块级元素
>
> li：块级元素

```html
<style>
  ul {
    margin: 0;
    padding: 0;
    list-style: none;
  }
</style>

<ul>
  <li>a</li>
  <li>b</li>
  <li>c</li>
  <li>d</li>
</ul>
```

## 表单元素 5

> 用来输入数据的是表单元素

### 输入框 5

```html
<input type="text" placeholder="请输入" value="hello world" />
```

### 密码框 5

```html
<input type="password" placeholder="请输入" value="hello world" />
```

### 复选框/多选 5

> name是用来和兄弟联动的
>
> id在复选框有特殊作用：和label联动

```html
<input type="checkbox" name="x" id="a" checked />
<label for="a">A</label>
<input type="checkbox" name="x" id="b" checked />
<label for="b">B</label>
<input type="checkbox" name="x" id="c" />
<label for="c">C</label>
```

### 单选框 5

> name是用来和兄弟联动的
>
> id在复选框有特殊作用：和label联动

```html
<input type="radio" name="y" id="aa" checked />
<label for="aa">A</label>
<input type="radio" name="y" id="bb" />
<label for="bb">B</label>
<input type="radio" name="y" id="cc" />
<label for="cc">C</label>
```

### 文本域 2

```html
<textarea placeholder="请评论">
	hello world
</textarea>
```

### 下拉选择框 5

```html
<select>
  <option>a</option>
  <option>b</option>
  <option>c</option>
</select>
```

### 按钮 5

```html
<button>第一种按钮</button> <input type="submit" value="第二种按钮" />
```

## 表格 5

```html
<!-- 表格容器 table -->
<table>
  <!-- 表头 table head -->
  <thead>
    <!-- 表格行 table row -->
    <tr>
      <th>第一列</th>
      <th>第二列</th>
      <th>第三列</th>
    </tr>
  </thead>

  <!-- 表身体 table body -->
  <tbody>
    <tr>
      <td rowspan="3">第一行第一格</td>
      <td>第一行第二格</td>
      <td>第一行第三格</td>
    </tr>
    <tr>
      <!-- <td>第二行第一格</td> -->
      <td>第二行第二格</td>
      <td>第二行第三格</td>
    </tr>
    <tr>
      <!-- <td>第三行第一格</td> -->
      <td>第三行第二格</td>
      <td>第三行第三格</td>
    </tr>
  </tbody>
</table>
```
